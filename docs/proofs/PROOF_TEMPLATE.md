# Proof File Template

This template defines the exact structure, writing style, and approach for creating taxonomy proof files. Every proof file must follow this template precisely. Do not deviate from the voice, structure, or reasoning depth demonstrated here.

---

## Writing Style Requirements

### Voice and Tone

- **Declarative and confident.** State classifications as reasoned conclusions, not suggestions. "The film is fundamentally character-driven" — not "The film could be considered character-driven."
- **Analytical, not academic.** Write like a thoughtful critic making a case, not a researcher citing theory. The prose should feel like essay writing — clear, muscular, purposeful.
- **No hedging.** Do not use "arguably," "perhaps," "it could be said that," or similar qualifiers. If the classification is correct, state it directly and support it with evidence.
- **No bullet points.** The entire proof file is written in flowing prose paragraphs. No lists, no bullet points, no shorthand. Every idea receives full sentence treatment.
- **No comparison to other media.** Each proof file must stand entirely on its own merits. Do not reference other films to explain why this film is classified a certain way. The reasoning must come from the film itself and the taxonomy definitions. If the reader later compares two proof files and notices differences, that is the reader's insight to arrive at — not the writer's to point out.
- **No em dashes.** Do not use — in any form. Restructure the sentence, use a colon, or use a comma instead. Parenthetical asides should be absorbed into the prose, not set off with punctuation.
- **Concrete evidence over abstract claims.** Every classification must be supported by specific scenes, character decisions, narrative structures, or thematic elements from the film. Do not make general statements without grounding them in the actual work.

### Paragraph Structure

- Paragraphs are substantial — typically 4-8 sentences. Short paragraphs are used only for emphasis or transition.
- Each paragraph develops a single idea fully before moving to the next.
- Paragraphs within a section build on each other. The first paragraph in a genre subsection establishes the claim; subsequent paragraphs deepen it with specific evidence and reasoning.

## Document Structure

Every proof file follows this exact structure, in this exact order. No sections are added, removed, or reordered.

### 1. Title

```
# [Film Title] ([Year])
```

The H1 heading is the film's title followed by its release year in parentheses. Nothing else appears on this line.

### 2. Opening Synopsis

Two paragraphs immediately below the title, with no heading or separator.

**First paragraph:** A narrative synopsis of the film — not a plot summary but a description of what the film is about at the story level. Written in present tense. Identifies the protagonist, the central situation, and the arc of the narrative without spoiler-level plot mechanics. This paragraph should convey the experience of the film, not just its events.

**Second paragraph:** The thematic thesis — one paragraph that distills what the film is fundamentally about beneath its surface. This is the emotional and philosophical core. It begins with "At its core, this is a film about..." or a variation that serves the same purpose. This paragraph frames everything that follows.

### 3. Horizontal Rule

```
---
```

A horizontal rule separates the synopsis from the classification sections.

### 4. Genre Section

```
## Genre: [Genre 1] · [Genre 2] · [Genre N]
```

The H2 heading lists all applicable genres separated by `·` (space-middot-space). There is no limit to the number of genres — a film may have one, two, three, or more. Every genre that passes the "remove it" test belongs in the heading. Genres are listed in order of narrative prominence.

**Opening paragraph:** A brief framing paragraph that explains how these genres interact in this specific film. Not a generic description of the genres — a specific statement about how they function together in this particular work. This paragraph establishes the relationship between the genres before individual analysis begins.

**Genre subsections:** Each genre gets its own H3 subsection:

```
### [Genre Name]
```

Each genre subsection contains 2-4 paragraphs of analysis. The analysis must:

- Ground the genre classification in the taxonomy's specific definition — reference core elements like "character-driven narrative," "speculative technology," "realistic human conflict," etc., using the terminology from the genre documentation
- Provide specific evidence from the film — character decisions, plot structures, thematic concerns — that demonstrate why this genre applies
- Explain what the genre is doing in this film, not just that it's present. How does it function? What does it contribute to the narrative that another genre would not?
- Where relevant, explain why the speculative/genre elements are narrative rather than aesthetic — if something could be mistaken for mere dressing, explain why it is structurally essential to the story

### 5. Horizontal Rule

```
---
```

### 6. Aesthetics Section

```
## Aesthetics: [Aesthetic 1] · [Aesthetic 2] · [Aesthetic N]
```

The H2 heading lists all applicable aesthetics separated by `·` (space-middot-space). There is no limit to the number of aesthetics — a film may have one, two, three, or more. Every aesthetic framework that demonstrably shapes the film's presentation belongs in the heading.

**Opening paragraph:** A brief framing paragraph that names the aesthetic frameworks and explains how they layer together in this film. Use the language of "presentation" and "framework" — aesthetics describe how the story is dressed, not what story is being told.

**Aesthetic subsections:** Each aesthetic gets its own H3 subsection:

```
### [Aesthetic Name]
```

Each aesthetic subsection contains 2-3 paragraphs of analysis. The analysis must:

- Identify the specific visual, cultural, and tonal elements from the film that match the aesthetic's definition
- Describe how the aesthetic manifests in concrete terms — settings, visual language, institutional structures, cultural frameworks, design choices
- Explain how the aesthetic serves as a presentation layer for the underlying genre narrative — it dresses the story without changing what story is being told
- Use the taxonomy's own terminology: "aesthetic framework," "visual language," "presentation layer," "dressing"

### 7. Horizontal Rule

```
---
```

### 8. Era Section

```
## Era: [Decade]s
```

The H2 heading names the production era (e.g., "2020s," "1980s").

**Opening paragraph:** Begin with a brief restatement of what the Eras taxonomy captures — "production-period storytelling DNA — not when a film is set, but how a film tells its story based on when it was made" — then name the specific era and its paradigm.

The Era section contains 2-3 paragraphs that analyze:

- The film's storytelling approach as a product of its production period — pacing, structure, audience assumptions, narrative confidence
- How the film reflects the era's paradigm as defined in the eras documentation, or, if the film falls in a decade not yet formally defined (e.g., 2020s), articulate the emergent paradigm the film represents
- Specific production and narrative choices that would not have been made in a different era — what makes this film a product of its time in how it tells its story

### 9. Horizontal Rule

```
---
```

### 10. Age Group Section

```
## Age Group: [Age Group]
```

The H2 heading names the age group (Kids, Family, Young Adult, or Mature).

The Age Group section contains 2-3 paragraphs that analyze:

- The storytelling paradigm the film operates within — reference the specific paradigm description from the age group documentation (e.g., "mature examination of complex themes without guaranteed resolution")
- Thematic complexity, emotional register, and moral framework as evidence for the classification
- How the film's approach to conflict resolution, moral ambiguity, and narrative closure fits the age group's paradigm
- If relevant, explain why the film does not belong to an adjacent age group by describing what it does that the other paradigm would not accommodate — without naming or belittling the other group, simply by demonstrating that this film's approach exceeds or differs from what the other paradigm assumes

## Classification Principles

### Taxonomy Fidelity

Every classification must be grounded in the Kaleidoscope taxonomy definitions as written. Before classifying any film, read the relevant genre singles, aesthetic definitions, era paradigms, and age group descriptions. Use the taxonomy's own language and core elements as the framework for analysis.

The taxonomy's central distinction — **genres describe the what, aesthetics describe the how** — must be respected absolutely. If a speculative element (e.g., space travel, advanced technology, prescience) is structurally essential to the narrative and removes the story's core conflict when removed, it is a genre element. If it provides setting, visual identity, or cultural framework but the core story could survive transplantation to a different context, it is an aesthetic element.

### The "Remove It" Test

For any element that could be classified as either genre or aesthetic, apply the removal test:

- Remove the element entirely. Does the core narrative conflict survive? If yes, the element is aesthetic. If no, the element is genre.
- Example: If spaceships and interstellar travel were removed from a film but replaced with, say, ocean voyages between continents, and the core story of political betrayal and personal identity remained intact — then Space Opera is an aesthetic, not a genre. But if prescience were removed and the protagonist's entire internal conflict vanished — then Science Fiction is a genre.

### Independence

Each proof file must be completely self-contained. No references to other films, other proof files, other versions of the same IP, or the reader's assumed knowledge of similar media. The classification must be justified entirely from the film's own content and the taxonomy's own definitions.

### Honesty Over Flattery

Classify what the film actually is, not what it aspires to be or what its marketing suggests. If a film marketed as Science Fiction is narratively an Action film in a Space Opera aesthetic, say so and explain why. The proof file exists to demonstrate that the taxonomy works — and the taxonomy only works if it is applied honestly.

## Pre-Writing Checklist

Before writing any proof file, read:

1. All relevant genre singles documentation (for every genre you intend to classify)
2. All relevant aesthetic documentation (for every aesthetic you intend to classify)
3. The eras index (for the production decade)
4. The age groups index (for the target paradigm)
5. The genres index and aesthetics index (for the genre-aesthetic distinction framework)

Do not write from memory of the taxonomy. Read the definitions fresh each time. Words mean things.

### Missing Genres or Aesthetics

If a film requires a genre or aesthetic that does not yet exist in the Kaleidoscope taxonomy, do not invent a classification or force the film into an ill-fitting existing category. Instead, pause the proof file and propose the new genre or aesthetic — name it, describe what it would cover, and explain why the film demands it. Wait for confirmation before proceeding. If the addition is approved, write the new genre single or aesthetic definition in the same format as the existing taxonomy documents and add it to the appropriate directory. Only then resume writing the proof file using the newly established definition.
